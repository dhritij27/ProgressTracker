# ProgressTracker

ProgressTracker is a cross-platform mobile app for tracking daily habits and tasks while visualizing productivity over time.  
It helps users build consistency, reflect on progress, and stay motivated through clear analytics and a calm, focused UX.

Built with **Expo + React Native**, HabitFlow runs on both **iOS and Android** from a single codebase.

---

## Features

### Habit Tracking
- Create and track daily habits
- Mark habits as completed per day
- Habit completion history stored by date

### Daily Tasks
- Create daily tasks and to-dos
- Check off tasks as completed
- Track task completion over time

### Calendar View
- View days with completed habits and tasks
- Understand productivity patterns by date

### Productivity Analytics
- Visualize productivity using charts
- Weekly productivity trends
- Habit vs task completion breakdown
- Percent-based productivity scoring

### Mobile-First Design
- iOS & Android support
- Touch-friendly UI
- Offline-first frontend architecture
